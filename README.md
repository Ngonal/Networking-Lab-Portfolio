# Computer-Networking-Lab-Portfolio
Hands-on lab portfolio showcasing practical experience in switching, routing, and security configurations, progressing from foundational setups to increasingly advanced and complex implementations. The Labs and their significance will be organized via the TCP/IP 5-Layer Network Model. A variety of networking simulators and emulators will be used depending on lab complexity. Emulator/Simulator software considered but not limited to Cisco Packet Tracer, EVE-NG, GNS3, and Cisco Modeling Labs
