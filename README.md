# Information
Hands-on lab portfolio showcasing practical experience in switching, routing, and security configurations, progressing from foundational setups to increasingly advanced and complex implementations. The Labs and their significance will be organized via the TCP/IP 5-Layer Network Model. A variety of networking simulators and emulators will be used depending on lab complexity.
