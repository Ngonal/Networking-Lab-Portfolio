# Layer 4 - Transport

## Core Function
Manages end-to-end communication reliability and multiplexing through port-based service addressing. It segments data streams, assigns source and destination port numbers to differentiate between concurrent application sessions, and—depending on the protocol—provides flow control, error recovery, and guaranteed delivery (TCP) or connectionless efficiency (UDP). This ensures that data arriving at the correct host is delivered to the correct process.

## Overview
The Transport Layer provides end-to-end communication services, including reliability, flow control, and multiplexing through port numbers. This section focuses on traffic filtering and address translation.
